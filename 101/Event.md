## SOC Metrics

- _Main Time to Detect (MTTD)_
  - Lower MTTD = Faster detection
- _Mean Time to Resolution (MTTR)_
  - Lower MTTR = More efficient IR (Instant Response)
- _Mean Time to Attend an Analyze (MTTA&A)_
  - Lower MTTA&A = reduced response latency
- _Incident Positive Rate_
  - Lower rate = more accurate detection
- _False Positive Rates (FPR)_
  - Lower rate = more accurate detection
- _Key Performance Indicators (KPIs)_
  - Measurable values to track performance
- _Key Risk Indicators (KRIs)_
  - Measurable values to access risk
- _Service level Agreements (SLAs)_
