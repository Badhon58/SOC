# Common THreats and Attacks

1. **Social Engineering**

- _Phishing_, _Spoofing_, _Vishing_, _SMiShing_, _Quishing_.
  - Exploiting humans
  - Spoofing
  - Phishing
    - Spear Phishing
    - Whaling
  - Vishing
  - SmiShing (SMS Phishing)
  - Quishing (QR Code Phishing)

2. **Malware**

- _Worm_, _Spyware_, _Adware Ransomware_, _Trojan_, _File less Malware_.
  - _Worm_
    - Self Replicating
    - Infect and propagate
    - Spreads across networks
      - Often Without requiring user interaction
    - Stuxnet, Blaster

  - _Spyware_
    - Monitor user activity
    - Display unwanted advertisements

  - _Trojan_
    - Disguised Malware
    - Acting as legitimate software
      - Games
      - Extensions
      - Productivity Applications
    - _RAT_ : Remote Access Trojan

  - _Botnets_
    - Network of compromised Devices
    - Network of compromised "Zombies".
    - Controlled by a remote attacker
    - Used to coordinate attacks
      - DDos
      - Spam Campaigns
      - Spread malware

  - _Ransomware_
    - Infection
    - Ransom Demand
    - Payment
    - Decryption

  - _File less Malware_
    - Memory -based malware
    - No traces on disk
    - Evade detection and logging
    - Living off the land
      - Powershell, WMI, Code Injection

3. **Identity and Account Compromise**
   - Usernames, passwords, SSN, PII
   - Impersonation, Fraud, Theft
   - Methods include
     - Phishing
     - Brute Force
     - Credential Stuffing
     - Social Engineering
     - Malware

4. **Insider Threats**
   - Threats from the "inside"
     - Current of former employees.
     - Contractors
     - Partners
   - Malicious, careless, or compromised
   - Can lead to severe exposure and damage
     - Data beaches.
     - Ip theft
     - Reputational damage
     - Legal issues

5. **Advanced Persistent Threats (APTs)**
   - Highly skilled, well funded adversaries
   - Sophisticated
   - Persistent
     - Long-term
   - Targeted

6. **Data Breaches**
   - Data exposure, theft, or compromise
   - PII credentials, financial records, IP
   - Malicious actions and human error
     - Misconfiguration
     - Inadequate security controls
   - Reputational damage, regulatory trouble

7. **Zero-Days**
   - Vulnerabilities previously unknown to a vendor
   - No Patches, no Mitigations
     - Zero days to patch
     - Zero days to protect
   - Risk mitigation, Risk Avoidance

8. **Supply Chain Attacks**
   - Exploits up the chain
     - Compromises security downstream
   - Suppliers, vendors, partners
   - Malware propagates down
     - Hard to detect
     - Malware from trusted entities
